# 📈 Tesouro Pipeline - Análise Automatizada do Tesouro Direto

## 🎯 O Problema
Investidores do Tesouro Direto perdem oportunidades porque não monitoram diariamente as oscilações de preço.

## 💡 A Solução
Um pipeline ETL completo que automatiza coleta, análise e alertas de oportunidades.
